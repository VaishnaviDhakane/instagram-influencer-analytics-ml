# Instagram Influencer Analytics & Impact Prediction

## Project Overview
This project analyzes Instagram influencer data to understand engagement
patterns and predict influencer impact using machine learning.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Key Features
- Data cleaning and preprocessing
- Feature engineering (likes per post, engagement strength)
- Exploratory Data Analysis (EDA)
- Machine learning model using Random Forest
- Business insights and recommendations

## Outcome
The project helps brands identify high-impact influencers using
data-driven insights instead of follower count alone.
